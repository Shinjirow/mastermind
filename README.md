# mastermind

プロジェクト演習課題2のやつ

## 使い方

これをcloneする

cloneした中にreidai01 ~ 03を入れる

```
mastermind/
|
|---Solver.java
|
|---Makefile
|
|---reidai01/
|   |
|   |---MasterMind.class
|   
|---reidai02/
|   |
|   |---MasterMind.class
|
|---reidai03/
    |
    |---MasterMind.class
```
こんな構造にしておいて、

`$ make`

してもらうと、3つ順番にサンプルケースを実行します
